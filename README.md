# Product Landing Page

## Description
A modern and responsive product landing page built using HTML, CSS, and JavaScript.
This project showcases a product with attractive UI design, responsive layout, smooth sections, and interactive elements.

## Features
- Responsive design
- Modern UI layout
- Interactive buttons and sections
- Smooth scrolling effects
- Mobile-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run

### Step 1: Download or Clone the Project

```bash
git clone <repository-link>
```

### Step 2: Open the Project Folder

Open the folder in VS Code or any code editor.

### Step 3: Run the Project

Simply open:

```plaintext
index.html
```

in your browser.
